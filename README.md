# UAV Topological Barcode ROI

Прототип алгоритма сопоставления визуальных ориентиров по топологическому баркоду ROI.

## Цель проекта

Разработать легкий интерпретируемый модуль для сопоставления области интереса (ROI) с базой эталонных ориентиров в задаче визуальной навигации БПЛА.

Алгоритм не является полной системой навигации БПЛА. Это отдельный модуль, который по входному ROI строит топологический дескриптор и ищет ближайшие эталоны в базе.

## Основная идея

Цепочка обработки:

```text
ROI → grayscale → пороги яркости → бинарные уровни → компоненты связности → топологический баркод → числовой дескриптор → Top-k поиск

## Итог

Для текущей версии лучшим режимом остается `none + quantile`. Простая фильтрация перед построением баркода не дала выигрыша по общей точности.
